About/Credits

Harvey-Mudd Miniature Machine (HMMM) is a simulated 16-bit, 24 instruction computer with 256 16-bit words of memory primarily targeted for educational use. HMMM was developed for Harvey Mudd's Introduction to Computer Science course and a full explanation of the machine can be found in the CS For All online text. Documentation for the language can be found on Harvey Mudd's website.

HMMM.js and the HMMM Simulator were built by Sean Hickey. Source code is available on GitHub.

HMMM.js is a Javascript implementation of a HMMM assembler and simulator written in the style of node.js.

The HMMM Simulator uses HMMM.js as a backend for assembly and simulation of the language. In addition, the HMMM Simulator uses several excellent open source projects:

AngularJS

Bootstrap

Ace Editor

FileSaver.js
